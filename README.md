

# 🍔 My Food App

A modern and responsive **food browsing web application** built with **React, Tailwind CSS, and DaisyUI**, powered by dynamic API data.

🔗 **Live Demo:** https://myfood-product.netlify.app/

---

## ✨ Features

- 🍽️ Browse delicious food items
- 🔍 Search and filter meals easily
- ⚡ Fast and responsive UI
- 🎨 Beautiful components using DaisyUI
- 🔄 Dynamic data from API
- 📱 Fully responsive (Mobile + Desktop)

---

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI

### Data & API
- 🌐 REST API integration
- 🔄 Dynamic data fetching

### Tools
- 🧑‍💻 VS Code
- 🔧 Git & GitHub
